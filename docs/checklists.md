# Чеклісти для тестування

## Модуль: Створення та налаштування навчального семестру (Варіант 8)

### Функціональність
- [ ] Створення нового семестру з валідними датами (початок < кінець).
- [ ] Можливість обрати робочі дні тижня (пн-пт/сб).
- [ ] Прив'язка існуючого шаблону дзвінків (пар) до нового семестру.
- [ ] Присвоєння статусу "Поточний" новому семестру.
- [ ] Автоматичне зняття статусу "Поточний" з попереднього семестру при активації нового.
- [ ] Використання "Дефолтного" семестру для автоматизації процесів.

### Негативні сценарії
- [ ] Спроба зберегти семестр, де дата завершення раніше дати початку.
- [ ] Введення однакової дати початку та кінця (семестр в 0 днів).
- [ ] Спроба створити семестр з назвою, яка вже існує (дублікат).
- [ ] Спроба залишити обов'язкові поля (дати, назва) порожніми.
- [ ] Створення семестру, що повністю накладається за датами на вже існуючий.

### UI/UX
- [ ] Відображення календаря (datepicker) для зручного вибору дат.
- [ ] Наявність зрозумілих повідомлень про помилки (наприклад, "Дата кінця не може бути в минулому").
- [ ] Кнопка "Зберегти" неактивна, поки не заповнені обов'язкові поля.
- [ ] Коректне відображення списку днів тижня (чекбокси розташовані рівно).
- [ ] Візуальне виділення "Поточного" семестру в загальному списку.
